## 17标准新特性
### 文件系统
* 迭代目录

```
#include <filesystem>

std::string dir_name = "";
for (const auto & entry :
std::filesystem::directory_iterator(dir_name)) {
    // 文件名 不包含后缀
    std::string filename = entry.path().stem();
    std::string file_path = entry.path().relative_path();
}

compile : --std=c++17 -lstdc++fs
```
